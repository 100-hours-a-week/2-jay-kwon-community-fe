### Say Anything Party
아무 말 대잔치 커뮤니

### Related Project
Backend API Server: [sap-api-server](https://github.com/juintination/sap-api-server)

### 사용 방법
1. Git clone repository

```
git clone https://github.com/juintination/say-anything-party.git

cd say-anything-party
```

2. Install modules

```
npm install
```

3. Run application

```
npm start
```
